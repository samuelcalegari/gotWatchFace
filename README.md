# GOT WatchFace for Garmin Watches

## Description ##

A GOT Styled WatchFace with Stark Logo for Garmin Watches
include steps goal graphic, second counter, battery  and heart rate informations

## Compatibles Devices ##

- fēnix® 5, quatix® 5
- fēnix® 5S fēnix® 5S
- fēnix® 5X fēnix® 5X, tactix® Charlie
